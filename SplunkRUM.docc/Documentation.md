# ``SplunkRum``

The Splunk RUM agent for iOS is a modular Swift package for Real User Monitoring. It provides several instrumentations to capture telemetry, helping you understand and improve your application's performance and user experience.

> :construction: This project is currently in **ALPHA**. It is **officially supported** by Splunk. However, breaking changes **MAY** be introduced.
