# ``SplunkAgent/SessionReplayModuleCustomId``

## Overview

The `SessionReplayModuleCustomId` protocol defines a public API for assigning custom identifiers to `UIView` elements. These identifiers can be used to uniquely identify UI elements in the session replay data for analysis and debugging.

## Topics

### View Custom ID

- ``subscript(view:)-44z77``
  Retrieves or adds a custom ID for the specified `UIView` instance.
- ``set(_:_:)-8g71h``
  Sets a custom identifier for the specified `UIView` instance.
