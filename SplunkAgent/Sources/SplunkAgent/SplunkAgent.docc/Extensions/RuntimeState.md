# ``SplunkAgent/RuntimeState``

## Overview

The `RuntimeState` object provides information about the current operational state and key configuration settings of the Splunk RUM agent during its runtime.

## Topics

### Agent Status

- ``status``
  The current operational status of the agent (e.g., running, not running).

### Configuration Details

- ``appName``
  The application name currently known to the agent.
- ``appVersion``
  The application version currently known to the agent.
- ``url``
  The URL endpoint currently used by the agent.
