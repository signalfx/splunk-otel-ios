# ``UIKit/UIView``

## Overview

Extensions to `UIView` that provide convenient access to Session Replay functionalities, allowing direct control over view sensitivity and custom identifiers.

## Topics

### Session Replay Sensitivity

- ``srSensitive``
  Element sensitivity for the specified `UIView` instance. Setting `nil` removes previously assigned explicit sensitivity.

### Session Replay Custom ID

- ``splunkRumId``
  Element custom identifier for the specified `UIView` instance. Setting `nil` removes previously assigned custom ID.
